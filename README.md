## Installation & Setup

1. **Clone the repository**:
   ```sh
   git clone https://github.com/zhbdripon/game-hub.git
   ```

2. **Navigate to the project root**:
   ```sh
   cd game-hub
   ```

3. **Install dependencies**:
   ```sh
   npm install
   ```

4. **Run the project**:
   ```sh
   npm run dev
   ```
